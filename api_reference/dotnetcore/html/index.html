Microsoft Azure IoT Gateway SDK - .NET Core Binding  1.0.3
The .NET Core binding allows modules written in any .NET language to run in a .NET Core environment
Microsoft Azure IoT Gateway SDK

Introduction

This document describes the .NET Core binding mechanism used by the Azure IoT Gateway SDK.

Features

  • Provides an interface to be implemented by .NET Core modules
  • Provides to modules a Broker for publishing messages to the gateway
  • Hosts modules in the .NET CoreCLR

Tested platforms

The .NET Core binding has been tested on the following platforms:

  • Windows 10
  • Ubuntu 14.04 LTS
  • Ubuntu 15.10
  • Ubuntu 16.04 LTS
