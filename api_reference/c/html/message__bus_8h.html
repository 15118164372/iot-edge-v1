Detailed Description

Library for configuring and using the gateway's message bus.

This is the API to create a reference counted and thread safe gateway message bus. The message bus broadcasts the messages to the subscribers. Messages on the message bus have a bag of properties (name, value) and an opaque array of bytes that is the message content.
