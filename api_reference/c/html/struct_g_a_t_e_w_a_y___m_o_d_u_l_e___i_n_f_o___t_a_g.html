VECTOR_HANDLE GATEWAY_MODULE_INFO_TAG::module_sources

A vector of pointers to GATEWAY_MODULE_INFO that this module will receive data from (link sources, this one being the sink).

If the handle == NULL this module receives data from all other modules.
