GATEWAY_HANDLE Gateway_Create_From_JSON ( const char *  file_path)

Creates a Gateway using a JSON configuration file as input which describes each module.

Parameters
file_pathPath to the JSON configuration file for this gateway. Sample JSON configuration file: 
{
    "modules" :
    [ 
        {
            "module name" : "foo",
            "module path" : "F:\\foo.dll",
            "args" : ...
        },
        {
            "module name" : "bar",
            "module path" : "F:\\bar.dll",
            "args" : ...
        }
    ],
    "links":
    [
        {
            "source": "foo",
            "sink": "bar"
        }
    ]
}
Returns
A non-NULL GATEWAY_HANDLE that can be used to manage the gateway or NULL on failure.
